# Schema Editor

A simple schema editor that allows adding tables, columns, and relationships via a visual editor.

## Usage

1. Run `python src/schema_editor.py` to start the editor.
2. Add tables, columns, and relationships using the `add_table`, `add_column`, and `add_relationship` methods.
3. Validate the schema using the `validate` method.
4. Export the schema as a migration file using the `export` method.
