# Analytics
A simple analytics system that tracks page views, unique visitors, session durations, and conversion events.

## Usage
1. Create an instance of the `Analytics` class.
2. Use the `add_page_view`, `add_unique_visitor`, `add_session_duration`, and `add_conversion_event` methods to track events.
3. Use the `get_page_views`, `get_unique_visitors`, `get_session_durations`, and `get_conversion_events` methods to retrieve analytics data.
4. Use the `export_to_csv` method to export analytics data to a CSV file.

## Testing
Run `pytest` to execute the tests.
